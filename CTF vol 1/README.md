# Mini-CTF na spotkanie otwierające
(C) Cyber Przem 2026
## Założenia

- 10 krótkich zadań z różnych obszarów cyberbezpieczeństwa.
- Czas: 60-90 minut.
- Praca indywidualna.
- Wszystkie zadania są lokalne i edukacyjne.
- Nie wolno wykonywać testów na cudzych systemach ani usługach internetowych.

| Nr | Kategoria | Materiał | Szacowany czas |

| 01 | Encoding | `01_encoding/message.txt` | 3-5 min |
| 02 | Linux/logi | `02_linux_log/access.log` | 3-5 min |
| 03 | Forensics/stego podstawowe | `03_forensics_strings/obraz.png` | 5-8 min |

## Minimalne narzędzia

- Linux / WSL / macOS Terminal
- Python 3
- Wireshark albo tshark
- Edytor tekstu
- Opcjonalnie: CyberChef, exiftool, binwalk
